安装

ACE&gperf

  1. 安装 GNU make 3.79.1 or greater
  2. 添加ACEROOT环境变量，值为解压后的ACE代码路径，如：

ACEROOT=/home/cs/faculty/schmidt/ACEwrappers; export ACEROOT

  1. 添加配置文件$ACEROOT/ace/config.h，插入如下内容：
  1. 添加配置文件 $ACEROOT/include/makeinclude/platformmacros.GNU, 添加如下内容：

include $(ACEROOT)/include/makeinclude/platformlinux.GNU :: 如果你需要安装ACE 到指定目录中，可以上面的文件中加上：INSTALLPREFIX = /usr/local(目标路径)

  1. 开始编译ACE，生成ACE到ACEROOT文件夹下的bin，lib,share,include 等目录中
  2. 在platformmacros.GNU文件中配置了INSTALLPREFIX，则可以使用make install 命令将ACE 安装到指定目录

安装TAO

  1. set TAOROOT=$ACEROOT/TAO
  2. 执行 make 在 $TAOROOT 目录
