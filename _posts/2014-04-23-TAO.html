安装

ACE&gperf

  1. 安装 GNU make 3.79.1 or greater
  2. 添加ACE_ROOT环境变量，值为解压后的ACE代码路径，如：

ACE_ROOT=/home/cs/faculty/schmidt/ACE_wrappers; export ACE_ROOT

  1. 添加配置文件$ACE_ROOT/ace/config.h，插入如下内容：

#include "ace/config-linux.h"

  1. 添加配置文件 $ACE_ROOT/include/makeinclude/platform_macros.GNU, 添加如下内容：

include $(ACE_ROOT)/include/makeinclude/platform_linux.GNU :: 如果你需要安装ACE 到指定目录中，可以上面的文件中加上：INSTALL_PREFIX = /usr/local(目标路径)

  1. 开始编译ACE，生成ACE到ACE_ROOT文件夹下的bin，lib,share,include 等目录中
  2. 在platform_macros.GNU文件中配置了INSTALL_PREFIX，则可以使用make install 命令将ACE 安装到指定目录

安装TAO

  1. set TAO_ROOT=$ACE_ROOT/TAO
  2. 执行 make 在 $TAO_ROOT 目录
